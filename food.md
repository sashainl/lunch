## :cyclone: 추천 로직 예시
- **짭짤한 + 뜨거운 + 국물 있는 + 푸짐하게 + 한식** → 김치찌개, 된장찌개  
- **매운 + 차가운 + 아삭한 + 특별하게 + 중식** → 마라샹궈, 냉짬뽕  
- **달콤한 + 부드러운 + 편안하게 + 양식** → 오므라이스, 크림 파스타
